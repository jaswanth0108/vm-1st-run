Question Import Format

Supported formats: .docx (Word) or .txt

1. Multiple Choice Question
1. Question text here?
a) Option 1
b) Option 2
c) Option 3
d) Option 4
Answer: b

2. Coding Question
2. Write a function to...
Input: 2 3
Output: 5
Constraints: 1 <= N <=100
Test Case 1:
Input: 4 5
Output: 9

3. Text Question
3. Explain the concept of recursion.
(No options or test cases needed)

* Questions must be numbered (1., 2., etc.)
* MCQ Options: a), b), c), d) or A., B., C., D.
* Answer format: "Answer: [option]"
* Coding keywords: Input:, Output:, Constraints:, Test Case [N]:
